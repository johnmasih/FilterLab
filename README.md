# 📸 FilterLab (Image Editing Web App)

A simple and interactive web-based image editor built using **HTML, CSS, and JavaScript**. This application allows users to upload images and apply various filters and adjustments in real-time using the **HTML5 Canvas API**.

---

## 🚀 Features

- Upload and preview images instantly  
- Apply filters like:
  - Brightness  
  - Contrast  
  - Saturation  
  - Blur  
  - Grayscale  
  - Sepia  
  - Hue Rotation  
- Real-time image editing using Canvas  
- Preset filter options (e.g., Normal, Drama, etc.)  
- Download edited image as PNG  
- Clean and responsive UI  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (Vanilla JS)** – Logic & interactivity  
- **Canvas API** – Image processing & rendering  

---

## 📂 How It Works

1. User uploads an image  
2. Image is rendered on the `<canvas>` element  
3. Selected filters are applied using JavaScript  
4. Canvas updates in real-time  
5. Final image can be downloaded using `toDataURL()`
